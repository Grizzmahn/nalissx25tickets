NALISS X 25 TICKET PURCHASE WEBSITE
-----------------------------------

A simple, mobile-friendly website for selling event tickets. Users can:
- View the event flyer
- Submit their details (name, email, phone)
- Select ticket quantity
- See payment instructions (Moniepoint)
- Send payment proof via WhatsApp

--------------------------------------------------
FILES INCLUDED
--------------------------------------------------
1. index.html           - Main website HTML file
2. IMG-20250717-WA0077.jpg - Event flyer image (must match exactly)

--------------------------------------------------
SETUP INSTRUCTIONS
--------------------------------------------------
1. REQUIREMENTS:
   - Text editor (VS Code recommended)
   - Modern web browser (Chrome, Firefox)
   - Netlify/GitHub account (for free hosting)

2. LOCAL SETUP:
   - Place both files in the same folder
   - Open index.html in browser to test
   - For live-reload testing in VS Code:
     * Install "Live Server" extension
     * Right-click index.html → "Open with Live Server"

3. CUSTOMIZATION:
   - Edit index.html to:
     * Change colors (modify <style> section)
     * Update payment details (search for "Moniepoint")
     * Change WhatsApp number (search for "2349080069065")
   - Replace IMG-20250717-WA0077.jpg with your flyer
     (keep filename EXACT or update HTML <img> tag)

--------------------------------------------------
DEPLOYMENT
--------------------------------------------------
1. NETLIFY (Easiest):
   - Drag-and-drop folder to https://app.netlify.com/drop
   - Wait for automatic deployment
   - Share the provided .netlify.app URL

2. GITHUB PAGES:
   - Upload files to new GitHub repository
   - Enable GitHub Pages in Settings → Pages
   - Select "main branch" as source

3. CUSTOM DOMAIN:
   - Available on both Netlify and GitHub Pages
   - Follow provider's instructions for DNS setup

--------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------
1. Image not loading:
   - Verify filename matches exactly (case-sensitive)
   - Check image is in same folder as HTML

2. Form not working:
   - Ensure all required fields are filled
   - Test in Chrome/Firefox (disable ad-blockers)

3. "Page Not Found" on Netlify:
   - Ensure main file is named index.html
   - Or access via /filename.html directly
